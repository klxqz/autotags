Шаблоны для страницы товара:
{$product} - объект(массив) товара
{$product.name} - название товара
{$product.summary} - краткое описание товара
{$product.description} - полное описание товара
{$product.price} - цена товара
{$product.min_price} - минимальная цена товара
{$product.max_price} - максимальная цена товара
{$product.compare_price} - зачеркнутая цена товара
{$product.count} - количество товара
{$product.features} - массив характеристик товара
{$category} - массив родительской категории
{$category.name} - название родительской категории
{$category.description} - описание родительской категории
{$path} - массив категорий пути товара
{$path[0]} - массив родительской категории(родитель 1-го уровня), аналогично {$category}
{$path[0].name} - название родительской категории, аналогично {$category.name}
{$path[0].description} - описание родительской категории, аналогично {$category.description}
{$path[1]} - массив родительской категории 2-го уровня
{$path[1].name} - название родительской категории 2-го уровня
{$path[1].description} - описание родительской категории 2-го уровня
{$path[2]} - массив родительской категории 3-го уровня
{$path[2].name} - название родительской категории 3-го уровня
{$path[2].description} - описание родительской категории 3-го уровня

Шаблоны для страницы категории:
{$category} - объект(массив) категории
{$category.name} - название категории
{$category.description} - описание категории
{$path} - массив родительских категорий до данной категории
{$path[0]} - массив родительской категории(родитель 1-го уровня)
{$path[0].name} - название родительской категории
{$path[0].description} - описание родительской категории
